Next we specif the specific driving seqeunce we want to simulate


\begin{DoxyCode}{0}
\DoxyCodeLine{n1=4}
\DoxyCodeLine{n2=2}
\DoxyCodeLine{kick\_building\_blocks = [ [[(\textcolor{stringliteral}{'x'},0.5),(\textcolor{stringliteral}{'dd'},1.0),(\textcolor{stringliteral}{'y'},0.4)],n1], [[(\textcolor{stringliteral}{'z'},0.5)],n2] ]}

\end{DoxyCode}


kick\+\_\+building\+\_\+blocks conatins all information on the sequence\+: a \textquotesingle{}x\textquotesingle{} kick with time 0.\+5 (in units of the energy scale J) is followed by an evolution with the dipolar hamiltonian (signaled by \textquotesingle{}dd\textquotesingle{}) with time 1.\+0 and a \textquotesingle{}y\textquotesingle{} kick with time 0.\+4. The whole sequence of \textquotesingle{}x\textquotesingle{} \textquotesingle{}dd\textquotesingle{} \textquotesingle{}y\textquotesingle{} is repeated n1 times. After that a \textquotesingle{}z\textquotesingle{} kick is applied n2 times. The whole sequence defines a single Floquet cycle.


\begin{DoxyCode}{0}
\DoxyCodeLine{c13\_dynamics = qnv.NV\_dynamics(c13\_spins,rabi\_freq,kick\_building\_blocks,detuning,AC\_function,noise)}

\end{DoxyCode}

\end{DoxyCode}


One can also write them directly using Qu\+SPin hamiltonian
