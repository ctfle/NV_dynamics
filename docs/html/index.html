Quantum Dynamics of Dipolar Coupled Nuclear Spins
Quantum Dynamics in dipolar coupled nuclear spins

Python package for (periodically) driven dipolar coupled nuclear spins. The package includes the class NV_system to create and design a system of dipolar coupled spins centered around a central NV center, and the class NV_dynamics for the time evolution of pure initial states of NV_system objects.

Installation via pip

Open a terminal window and navigate into the folder QNV4py. Type

pip install .

Alternatively: Open a terminal window, navigate into the folder QNV4py. Type

make install

To open the documentation file from the terminal: Open a terminal window and navigate to the folder QNV4py. Type

make show docu

Required packages

  • numpy, scipy
  • h5py, python package for .hdf5 data files (https://www.hdfgroup.org/solutions/hdf5/)
  • Quspin, for the construction of observables and Hamiltonians (https://weinbe58.github.io/QuSpin/)

How to use the code:

Check out the example files under 'Related Pages' (sidebar)!

The code consists of two classes: NV_system, to construct a random graph of atoms, and NV_dynamics to evolve in time with user defined sequence. To setup a working code, first you have to construct a NV_system object. Then, use it to build a NV_dynamics object. NV_system can be used with default parameter settings (for example C13_object = NV_system.default(L) ), which builds the random graph with default settings. An NV_dynamics object requires the following input:

  • nv_instance, an NV_system object
  • rabi_freq, the amplitude of the kicks
  • kick_building_blocks, the building blocks of the drive
  • detuning=None, Detuning (left over single particle field in the rotating frame, Default None)
  • AC_function=None, an AC field given as an arbitrary function, Default None
  • noise=None, some noise to increase ergodicity, Default None
